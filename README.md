# about

Angular directive to get and set date value from an input.

# how to install

```
npm install n4-dateInput-directive --save
```

or

```
bower install n4-dateInput-directive --save
```

# how to use

```
<n4-date-input data-ng-model="value"></n4-date-input>
```
